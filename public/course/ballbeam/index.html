Frequently Asked Questions

How do I set up my development environment?

Follow the Setup module in the course. You'll need to install PlatformIO extension in VS Code, then configure your board settings. If you encounter issues, submit a support ticket with your error messages.

My watch won't connect via USB. What should I do?

Try these steps: 1) Use a different USB cable (some cables are charge-only). 2) Try a different USB port. 3) Install the CH340 driver if on Windows. 4) Check Device Manager to see if the device is recognized.

How long do I have access to the course materials?

You have lifetime access to all course materials once purchased. Virtual support is available for 3 months after your kit delivery.

Can I get help if I'm stuck on a project?

Yes! Use the Support button to submit a ticket describing your issue. Include any error messages, screenshots, or code snippets. We typically respond within 24-48 hours. For complex issues, we may schedule a one-on-one video call.

How do I submit my final project?

Navigate to the Final Project Submission module at the end of the curriculum. You'll upload your code as a zip file, provide a video link demonstrating your project, and write a brief description.
